# Generador de Contraseñas en Python

Proyecto sencillo que genera **contraseñas aleatorias** utilizando letras, números y símbolos. Está pensado como un ejercicio práctico para reforzar el uso de **librerías estándar**, **bucles** y **manejo de entradas por consola** en Python.

---

## Descripción

El programa solicita al usuario la **longitud deseada de la contraseña** y genera una contraseña aleatoria combinando:

* Letras mayúsculas y minúsculas
* Dígitos numéricos
* Símbolos especiales

Cada ejecución produce una contraseña diferente.

---

## Funcionalidades

* Solicita la longitud de la contraseña
* Genera contraseñas aleatorias
* Combina diferentes tipos de caracteres
* Muestra la contraseña generada en pantalla

---

## Conceptos aplicados

* Importación de módulos (`random`, `string`)
* Uso de bucles
* Manejo de cadenas de texto
* Entrada y salida de datos por consola

---

## ▶️ Cómo ejecutar el programa

1. Asegúrate de tener Python instalado (versión 3.x).
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta el siguiente comando:

```bash
python password.py
```

4. Ingresa la longitud deseada de la contraseña cuando el programa lo solicite.
